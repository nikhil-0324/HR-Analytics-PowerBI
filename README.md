📊 HR Analytics Dashboard | Power BI 

📌 Project Overview 

This project is an interactive HR Analytics Dashboard built using Microsoft Power BI. The dashboard helps HR teams analyze employee attrition, workforce distribution, and employee demographics to support data-driven decision making. 

 

🎯 Objectives 

Analyze employee attrition trends 

Identify departments with high turnover 

Understand attrition by job role, age group, gender, and salary group 

Create an interactive dashboard for HR managers 

 

🛠️ Tools & Technologies 

Microsoft Power BI 

Power Query 

DAX (Data Analysis Expressions) 

CSV Dataset 

 

📂 Dataset 

The dataset contains HR employee information, including: 

Employee ID 

Age 

Gender 

Department 

Job Role 

Monthly Income 

Business Travel 

Education 

Attrition 

Job Satisfaction 

Work-Life Balance 

Years at Company 

and other HR-related attributes. 

 

📊 Dashboard KPIs 

👥 Total Employees

📈 Dashboard Visualizations 

Attrition by Department 

Active Employees by Department 

Attrition by Job Role 

Attrition by Gender 

Attrition by Age Group 

Attrition by Salary Group 

 

🎛️ Interactive Filters 

The dashboard includes slicers for: 

Department 

Gender 

Job Role 

These allow users to interactively filter all dashboard visuals. 

 

💡 Key Business Insights 

Research & Development has the largest workforce. 

Laboratory Technicians experience the highest employee attrition. 

Employees in the lower salary group have the highest turnover. 

Employees aged 26–35 show the highest attrition. 

The dashboard enables HR managers to identify workforce trends and areas requiring attention.
